# 소노 GS팀 프로젝트 — 노션 활용 가이드

## 1. CSV 파일을 노션 데이터베이스로 가져오기

### Step 1: 노션에서 새 페이지 생성
- 노션 좌측 사이드바에서 `+ 새 페이지` 클릭
- 페이지 제목: **소노 GS팀 대시보드 프로젝트**
- 페이지 아이콘: 📊 (선택사항)

### Step 2: CSV Import
1. 페이지 본문에서 `/table` 입력 → **Table - Full page** 선택
2. 우측 상단 `···` 메뉴 → **Merge with CSV** 클릭
3. `sono_project_notion.csv` 파일 업로드
4. 미리보기에서 컬럼 매핑 확인 후 **Import** 클릭

> **팁:** Import 후 컬럼 타입이 자동 설정되지 않을 수 있습니다. 아래 속성 설정을 따라 수동 조정하세요.

|--------|------|------|
| **ID** | Title | 기본 제목 속성 (자동) |
| **업무명** | Text | — |

**상태:**
- 완료 → Green
- 진행중 → Blue

- 부분완료 → Orange
- 보류 → Gray

**우선순위:**
- 🔴 긴급 → Red

**카테고리:** 각각 다른 색상 지정 (A=Blue, B=Purple, C=Pink, D=Green, E=Yellow, F=Orange, G=Gray, H=Brown)

## 3. 칸반(Board) 뷰 만들기

### 상태별 칸반 보드 (메인 뷰)

1. 테이블 좌측 상단 `+ Add a view` 클릭
2. **Board** 선택, 이름: "상태별 칸반"
3. **Group by** → `상태` 선택
4. 컬럼 순서를 드래그로 정렬: `진행중` → `대기` → `부분완료` → `보류` → `완료`
5. 완료 컬럼은 기본 접기(Hide) 설정하면 깔끔함

> 이렇게 하면 각 업무 카드를 드래그해서 상태를 변경할 수 있습니다.

### 카테고리별 칸반 보드

1. `+ Add a view` → **Board**, 이름: "카테고리별"
2. **Group by** → `카테고리` 선택
3. 각 카테고리(A~H)가 컬럼이 되어, 프로젝트 영역별로 업무 현황을 한눈에 볼 수 있음

### 우선순위별 칸반 보드

1. `+ Add a view` → **Board**, 이름: "우선순위별"
2. **Group by** → `우선순위` 선택
3. 긴급/높음 업무가 바로 눈에 들어옴

---

## 4. 추가 뷰 설정

### 타임라인 뷰 (완료일 기준)

1. `+ Add a view` → **Timeline**
2. Date 속성: `완료일` 선택
3. 완료된 업무의 시간 흐름을 시각적으로 확인 가능

### 갤러리 뷰 (카테고리 요약)

1. `+ Add a view` → **Gallery**
2. 카드 미리보기에 `카테고리`, `상태`, `우선순위` 표시
3. 각 업무의 핵심 정보를 카드 형태로 확인

---

## 5. 필터 & 정렬 추천

### 자주 쓰는 필터

**"지금 해야 할 일" 필터:**
- `상태` is not `완료`
- `상태` is not `보류`
- Sort: `우선순위` Ascending

**"이번 달 완료" 필터:**
- `완료일` is `This month`
- `상태` is `완료`

**"카테고리별 미완료" 필터:**
- `상태` is not `완료`
- Group by: `카테고리`

## 6. 노션답게 활용하는 팁

### 업무 카드에 상세 내용 추가
각 행(업무)을 클릭하면 페이지로 열립니다. 여기에 아래 내용을 추가하세요:
- 관련 스크립트/파일 링크
- 스크린샷
- 서브 체크리스트 (예: C-12 다올 PMS DB 연동의 세부 단계)
- 코멘트로 진행 상황 기록

### 리스크 관리 섹션 별도 DB
원본 파일의 "리스크 항목"은 별도 데이터베이스로 만들어 관리하세요:
- 속성: 리스크명 / 상태 / 심각도(높/중/낮) / 대응방안 / 관련 업무(Relation)
- 관련 업무 속성을 Relation으로 설정하면 프로젝트 DB와 연결됩니다

### 대시보드 페이지 구성
프로젝트 메인 페이지에 아래 요소를 배치하면 한눈에 현황 파악이 가능합니다:
- **Linked view 1:** 상태별 칸반 (진행중/대기만 필터)
- **Linked view 2:** 우선순위 긴급/높음만 필터링한 테이블
- **Callout 블록:** 현재 우선순위 TOP 5 (원본의 우선순위 섹션 내용)
- **Toggle 블록:** 파일 구조 / 세션 관리 팁 등 참고 정보

### 아이콘 & 커버
- 페이지 아이콘: 📊 또는 🏨
- 카테고리별 아이콘: A=🔍 B=📈 C=🏢 D=📦 E=🌏 F=🎨 G=⚙️ H=🔗
- 커버 이미지: 소노 리조트 관련 이미지
